# YS Knife UI ElementPlus 示例项目

这个目录包含了YS Knife UI ElementPlus组件库的使用示例，帮助开发者了解如何使用组件库中的各个组件。

## 运行示例

```bash
# 进入example目录
cd example

# 使用vite启动开发服务器
npx vite
```

服务器启动后，将在浏览器中打开示例页面，展示各个组件的使用方法。

## 示例内容

- YsButton 组件示例
- YsTable 组件示例

## 目录结构

```
- index.html      # HTML入口文件
- main.js         # 应用入口文件
- App.vue         # 示例应用组件
- vite.config.js  # Vite配置文件
```

## 注意事项

这个目录仅用于开发和演示，不会被包含在最终的npm包中。
